# TimeLine

| 属性        | 说明                          | 类型            | 默认值         |
| --------- | --------------------------- | ------------- | ----------- |
| data | data是K线图的数据来源，数据结构必须为`{"lastPrice":29.85,"avgPrice":30.03,"y":1095,"x":"14:24","color":"green"}` | array        |  null |

| amOpenL  |  是否禁用                         | 	bool | 无           |
| yExtents  |  是否禁用                         | 	bool | 无           |
